ما هي الحوسبة السحابية؟

الحوسبة السحابية هي تقديم موارد تقنية مثل الخوادم، قواعد البيانات، التخزين، والشبكات عبر الإنترنت (السحابة) بدلًا من تشغيلها محليًا على جهازك أو خوادم داخلية.

لماذا تُستخدم؟

  • توفير في التكلفة والوقت
  • قابلية التوسع حسب الطلب
  • المرونة وسهولة النشر

أنواع النشر

  • سحابة عامة (Public)
  • سحابة خاصة (Private)
  • سحابة هجينة (Hybrid)

نظرة عامة تفصيلية

تتضمن الحوسبة السحابية ثلاثة نماذج خدمة رئيسية: SaaS، PaaS، IaaS. كما تعتمد على مراكز بيانات مُدارة وموزعة لتقديم موارد افتراضية قابلة للتخصيص.

IaaS — البنية التحتية كخدمة

توفر آلات افتراضية، شبكات، وتخزين قابل للتخصيص (مثال: إنشاء خادم VPS على السحابة).

PaaS — منصة كخدمة

تتيح للمطورين نشر التطبيقات دون القلق على البنية التحية (مثال: منصات تطبيقات مُدارة).

SaaS — البرنامج كخدمة

تطبيقات جاهزة للاستخدام عبر المتصفح أو التطبيقات (مثال: بريد إلكتروني مُدار، أدوات تعاون).

خدمات سحابية شائعة

  • التخزين: S3، Blob Storage، File Storage.
  • حسابات افتراضية: Virtual Machines، EC2.
  • شبكات: VPC، Load Balancer، DNS مُدار.
  • قواعد بيانات مُدارة: RDS، Cloud SQL، NoSQL managed.
  • خدمات دون خادم: Functions، Serverless.

مثال عملي — نشر صفحة ويب بسيطة (مخطط)

<!-- خطوات سريعة -->
1. أنشئ حسابًا على مزود سحابي.
2. أنشئ وحدة تخزين ورفع الملفات.
3. أنشئ CDN إن احتجت أداءً أفضل.
4. اربط اسم نطاقك.

الأمن والحوكمة

الأمن في السحابة يتضمن: تشفير البيانات (ساكنة ومنقولة)، إدارة الهوية والصلاحيات (IAM)، النسخ الاحتياطي، وسياسات الوصول الشبكي.

  • تفعيل المصادقة متعددة العوامل (MFA)
  • تجنُّب توفير مفاتيح الوصول في الشيفرة
  • مراجعة السجلات بشكل دوري

مزودون مشهورون

Amazon Web Services (AWS)

يضم مجموعة كبيرة من الخدمات ومتدرج حسب الاحتياج.

Microsoft Azure

متكامل مع خدمات مايكروسوفت والشركات الكبرى.

Google Cloud Platform (GCP)

قوي في تحليلات البيانات والذكاء الاصطناعي.

مصطلحات أساسية

  • VM: آلة افتراضية
  • Container: وحدة تشغيل خفيفة مثل Docker
  • Region / Zone: مواقع جغرافية لمراكز البيانات
  • Autoscaling: زيادة/تقليل الموارد تلقائيًا عند الحاجة

مصادر ومشاريع مقترحة

  • دورات مجانية للمبتدئين عن AWS/GCP/Azure
  • مشروع عملي: أنشئ موقعًا ثابتًا واستضفه على S3 + CloudFront (CDN)
  • مشروع متقدم: بناء API بسيط باستخدام Functions وقاعدة بيانات مُدارة

هل تريد أن أحول أي مشروع من هذه الأفكار إلى دليل خطوة-بخطوة؟
